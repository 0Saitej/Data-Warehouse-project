**2. Project Objectives**

 ~ Design a structured data warehouse schema (Star or Snowflake Schema).

 ~ Build fact and dimension tables.
 
 ~ Enable efficient querying for business insights.
 
 ~ Create dashboards and reports for analysis.

**Data Sources**

~ Temporary storage for extracted data.

~ Data cleaning and preprocessing.

**Data Warehouse Layer**

~ Fact Tables.

~ Dimension Tables.

~ Optimized structure for analytical queries.

**Analytics and Reporting Layer**

~ The warehouse is designed using Dimensional Modeling.

Dimension Tables

~ Time Dimension.

~ Customer Dimension.

~ Product Dimension.

~ Location Dimension.
